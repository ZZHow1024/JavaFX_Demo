## 使用说明

- 确定您使用的操作系统
    - Windows：
        - 选择 .zip压缩包/.exe安装包/.msi安装包
    - Linux：
        - 选择 .deb安装包/.rpm安装包
    - macOS：
        - 选择 .dmg磁盘镜像/.pkg安装包
- 下载对应的文件
- Windows 可直接运行 .zip 压缩包中的 .exe 可执行程序或选择 .exe 安装包与 .msi 安装包执行安装操作，Linux 和 macOS 需要执行安装操作后再运行

## 功能介绍

- MagicCalculator1.0.0
    - 两个数之间的四则运算
    - 两个数之间的模运算
    - 支持计算小数
